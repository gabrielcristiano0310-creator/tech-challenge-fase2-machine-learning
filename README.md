# 🍷 Tech Challenge – Fase 2

## Classificação da Qualidade de Vinhos utilizando Machine Learning

Projeto desenvolvido como parte da Pós-Tech em Data Analytics da FIAP.

---

## 📌 Objetivo

Este projeto tem como objetivo desenvolver e comparar modelos de Machine Learning capazes de classificar vinhos em duas categorias (Alta Qualidade e Baixa/Média Qualidade), utilizando características físico-químicas presentes na base WineQT.

---

## 📊 Base de Dados

- WineQT Dataset
- 1.143 registros
- 11 variáveis preditoras
- Variável alvo: Quality

---

## 🛠 Tecnologias

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## 🤖 Modelos Utilizados

- Regressão Logística
- Random Forest

---

## 📈 Resultados

| Modelo | Acurácia |
|---------|---------:|
| Regressão Logística | **90,83%** |
| Random Forest | **91,70%** |

O Random Forest apresentou melhor desempenho para o problema proposto.
